## Client Brief
A mental health coach wants an app that encourages daily journaling and mood tracking.

## Key Requirements

### 1. Entries
- Create daily entries with:
  - Date
  - Mood (Happy, Neutral, Sad, Anxious, Angry, etc.)
  - Text note
  - Optional tags (Work, Family, Health, etc.)
- Mark entries as **Important** or **Favorite**
- Edit and delete entries

### 2. Views
- Week/Month overview of mood trends (simple list with mood colors)
- Filter entries by:
  - Mood
  - Tag

### 3. Insights
- Show stats for last **7/30 days**:
  - Most common mood
  - Most common tag
- Display a short **summary message** based on mood patterns

### 4. Extras
- **Daily prompt:** Show a rotating pre-defined journaling question
- **Export:** Export last **7/30 entries** as text
